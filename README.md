# 🎮 Шмарио

Пиксельный платформер-приключение в стиле классического Mario.

## 🌍 Играть онлайн
**[Начать игру →](https://madiyarbayern-ship-it.github.io/Smario/)**

## 🏙 4 уровня по городам мира
- 🌷 **Амстердам** — каналы и велосипеды
- 🏝 **Пхукет** — пляж и кокосы
- 🐉 **Пекин** — драконы и пагоды
- 🏔 **Алматы** — горы и яблоки

## 🎯 Управление
- **← → / A D** — движение
- **↑ / W / Пробел** — прыжок
- На мобильных — кнопки на экране

## ✨ Особенности
- Уникальные враги в каждом городе
- Прыжок на голову врага убивает его (Mario-style)
- 3 жизни в виде сигарет 🚬
- Финиш — самолёт ✈️ или красная спортивная машина 🏎 (Алматы)
- Интро музыка на стартовом экране
